# 🚶 Smart PIR Motion Detection System using ESP8266

## 📌 Project Overview

The Smart PIR Motion Detection System is an IoT-based security project developed using ESP8266 NodeMCU and a PIR (Passive Infrared) sensor. The system detects human movement in real time and displays the motion status on a web dashboard. When motion is detected, an LED indicator is activated and the dashboard updates instantly.

## 🚀 Features

- Real-Time Motion Detection
- PIR Sensor-Based Monitoring
- ESP8266 Web Dashboard
- Motion Detected Status Display
- LED Alert Indication
- Wi-Fi Enabled Monitoring
- Automatic Dashboard Updates

## 🛠️ Hardware Components

- ESP8266 NodeMCU
- PIR Motion Sensor (HC-SR501)
- LED
- Resistor
- Breadboard
- Jumper Wires
- USB Cable

## 🔌 Circuit Connections

| Component | ESP8266 Pin |
|-----------|------------|
| PIR OUT | D4 |
| LED | D5 |
| VCC | 3.3V |
| GND | GND |
